- Implemented web ontologies and RO-Crate standards with JSON-LD metadata to structure datasets, contextual entities, and documentation, enhancing reproducibility, interoperability, and data sharing across 20+ research projects.

- Automated data validation and transformation workflows using Python and SQL, creating rule-based checks to identify duplicates, missing fields, and inconsistencies, reducing recurring data quality issues by 30%.

- Established version-controlled workflows using Git and structured documentation to support reproducible, collaborative research and long-term project scalability.

- Enabled structured analysis of large-scale unstructured datasets, significantly improving research efficiency and standardizing outputs for 100+ researchers.
